# Support

Termfolio is maintained on a best-effort basis. Clear, reproducible reports are welcome, but no response-time or resolution-time guarantee is provided.

## Before asking for help

1. Check the [README](README.md) and detailed [configuration](docs/configuration.md), [content](docs/content.md), and [development](docs/development.md) guides.
2. Search existing issues for the same question or behaviour.
3. Record the Termfolio version or commit, complete `hugo version` output, and the smallest configuration or content example that reproduces the problem.
4. Confirm whether the behaviour also occurs with custom layouts and styles temporarily removed.

## Choose the right channel

| Need | Channel |
| --- | --- |
| Help using or configuring Termfolio | [Usage question](https://github.com/joe-mccarthy/termfolio/issues/new?template=question.yml) |
| Reproducible Termfolio defect | [Bug report](https://github.com/joe-mccarthy/termfolio/issues/new?template=bug_report.yml) |
| New behaviour or design proposal | [Feature request](https://github.com/joe-mccarthy/termfolio/issues/new?template=feature_request.yml) |
| Share a public Termfolio site | [Showcase submission](https://github.com/joe-mccarthy/termfolio/issues/new?template=showcase.yml) |
| General Hugo usage unrelated to the theme | [Hugo community support](https://discourse.gohugo.io/) |
| Contribute code or documentation | [Contributing guide](CONTRIBUTING.md) |

Issue discussions are public. Do not include credentials, private site content, access tokens, personal data, or other sensitive information.

## Scope

Support covers the latest published Termfolio release and the supported Hugo range documented in the README. Maintainers may close questions caused by unsupported Hugo versions, unmaintained third-party overrides, or behaviour outside the theme after identifying the appropriate upstream project or documentation.
